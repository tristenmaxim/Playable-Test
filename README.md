# Runner Game - Playable Developer Test

Клон игры Runner (бесконечный раннер) на основе референса.

## Структура проекта

```
├── index.html              # HTML для разработки
├── game.html               # Старая версия (будет удалена)
├── src/                    # Исходный код
│   ├── config.js           # Конфигурация игры
│   ├── game/               # Игровая логика
│   │   ├── Game.js         # Главный класс
│   │   ├── Background.js   # Фон с параллаксом
│   │   └── Player.js       # Игрок
│   ├── ui/                 # UI элементы (будет добавлено)
│   ├── audio/              # Аудио система (будет добавлено)
│   └── utils/              # Утилиты (будет добавлено)
├── assets/                 # Исходные ассеты
│   ├── images/             # Изображения
│   ├── audio/              # Аудио файлы
│   └── fonts/              # Шрифты
├── scripts/                # Скрипты сборки (будет добавлено)
├── reference_assets/       # Референсные ассеты
└── PLAN.md                 # План разработки
```

## Разработка

1. Откройте `index.html` в браузере (нужен локальный сервер для модулей)
2. Или используйте Live Server в VS Code
3. Код в `src/` - модульная структура для удобной разработки
